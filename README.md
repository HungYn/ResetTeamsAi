# 🧹 Microsoft Teams 帳號清除工具

在共用電腦上使用 Microsoft Teams 時，常因殘留帳號資訊導致自動登入、同步異常或資安風險。本工具提供一鍵清除 Teams 登入資訊與快取的功能，協助使用者快速重置 Teams 狀態，提升安全性與操作效率。

## 🎯 功能與價值

- 🔒 **清除殘留登入資訊**  
  一鍵移除 Teams 帳號、密碼與快取，避免自動登入與帳號殘留。

- 🛡️ **提升資安**  
  防止共用電腦上他人誤用或惡意存取前一位使用者的帳戶。

- 🔄 **簡化交接流程**  
  更換使用者前快速重置 Teams 狀態，節省 IT 支援與現場處理時間。

- 🧯 **降低錯誤與疑難排解**  
  避免因快取損毀或舊設定導致的登入循環、同步異常。

- 📋 **統一作業標準**  
  提供標準化清理流程，減少人工刪檔與操作風險。
